# AstrA Assistant

AstrA is a persistent, local-first thinking partner designed to help me understand, learn, and plan complex systems.

## What AstrA is
- A reasoning-first assistant with disciplined memory
- A planner that turns ideas into executable next steps
- A teacher that explains with clear mental models and examples

## What AstrA is not
- Not a generic chatbot
- Not an autonomous agent swarm
- Not an infinite conversation archive

## Core modes
- Understand: clarify concepts, reduce confusion, correct assumptions
- Learn: teach step-by-step, build durable mental models
- Plan: generate constraints-aware plans, tradeoffs, and next actions
